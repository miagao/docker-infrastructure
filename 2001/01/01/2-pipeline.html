# Continuous Delivery Pipeline ![](./assets/continuous-deployment-pipeline.png) -- ## What is Docker Docker is an open platform for developers and sysadmins to build, ship, and run distributed applications. - Docker cli - Dockerfiles - Docker registry -- ## Docker advantages for Continuous Delivery - Faster - slow one time events happen only once on image creation, not on instance creation. Image creation is fast and cheap. - Better: Same images for development/testing/production. Portability and scalability. - Workflow: develop/ unit test Dockerfiles / commit / post-commit hooks triggers new delivery / build new images from source/ the image is the software artifact / system tests run on these images. - Encourages collaboration between Dev and Ops. - Version Control of images. - Share or reuse Docker Hub images.
